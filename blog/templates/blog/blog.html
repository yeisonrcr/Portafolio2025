Explicación del Código

TensorFlow es una biblioteca para construir y entrenar modelos de aprendizaje automático.
En esta sección, proporcionamos una explicación detallada y clara del código que se muestra a continuación.
Nuestro objetivo es hacer que conceptos técnicos complejos sean fáciles de entender.

  • Fashion MNIST es un conjunto de datos con imágenes de ropa.
  • Sequential, Dense, Flatten son herramientas de TensorFlow para construir redes neuronales.
  • Adam es un algoritmo que ayuda al modelo a aprender.
  • NumPy es una biblioteca para trabajar con matrices y números.


El código define una función create_and_train_model() que utiliza TensorFlow para entrenar un modelo de red neuronal en el conjunto de datos de moda MNIST, que consiste en imágenes de ropa. Primero, carga los datos y normaliza las imágenes dividiendo los valores de píxeles por 255 para que estén en el rango de 0 a 1. Luego, crea un modelo secuencial con una capa Flatten que aplana las imágenes 2D en vectores 1D, una capa densa con 128 neuronas utilizando la función de activación ReLU, y una capa de salida con 10 neuronas usando softmax para la clasificación. El modelo se compila con el optimizador Adam y la pérdida de entropía cruzada categórica esparsa. Después, el modelo se entrena durante 10 épocas con los datos de entrenamiento y se valida con los datos de prueba. Finalmente, se evalúa el modelo con los datos de prueba y se imprime la precisión obtenida. Al finalizar, la función retorna el modelo entrenado.

Código Fuente



import tensorflow as tf
from tensorflow.keras.datasets import fashion_mnist
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense, Flatten
from tensorflow.keras.optimizers import Adam
import numpy as np

def create_and_train_model():
    # Cargar el conjunto de datos de moda MNIST
    (x_train, y_train), (x_test, y_test) = 
        fashion_mnist.load_data()

    # Normalizar los datos
    x_train, x_test = x_train / 255.0, x_test / 255.0

    # Crear el modelo
    model = Sequential([
        Flatten(input_shape=(28, 28)),     # Aplana las imágenes de 28x28 píxeles
        Dense(128, activation='relu'),     # Capa densa con 128 neuronas y activación ReLU
        Dense(10, activation='softmax')    # Capa de salida con 10 neuronas y activación softmax
    ])

    # Compilar el modelo
    model.compile(optimizer=Adam(),
                  loss='sparse_categorical_crossentropy',
                  metrics=['accuracy'])

    # Entrenar el modelo
    model.fit(x_train, y_train, epochs=10, 
        validation_data=(x_test, y_test))

    # Evaluar el modelo
    test_loss, test_acc = model.evaluate(x_test, y_test, verbose=2)
    print('\nTest accuracy:', test_acc)

    return model

# Ejecutar la función
if __name__ == "__main__":
    model = create_and_train_model()

git clone x :clonar el git

git checkout main : rama main

git pull origin main : descargo ultimos cambios del repositorio remoto

git merge main : fusionar cambio en el main con mi rama 

git checkout -b yeison : creo una rama y automaticamente ingreso 

git branch : ver que rama estoy


 git push -u origin yeison  : push al github , rama usada, aun no tiene el pull request del jefe pero esta subido para revision
